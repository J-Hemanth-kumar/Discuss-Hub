# Discuss-Hub
A simple and user-friendly online discussion platform where people can create topics, share ideas, and interact through conversations. It provides secure login, easy navigation, and organized discussions, making it a convenient space for communities to connect and communicate effectively.

---

## 🚀 Features

- User registration and login
- Create discussion topics
- Post and reply to conversations
- Password reset via email
- Edit posts
- Clean and responsive interface

---

## 💡 What this project does

Discuss Hub allows users to engage in meaningful discussions by creating topics and replying to others. It is designed to be simple, fast, and easy to use for everyday communication.
